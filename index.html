Nostr Web Components デモ

使い方ガイド

基本的なセットアップ

Nostr Web Componentsは軽量で使いやすいWebコンポーネントライブラリです。Nostrプロトコル機能をWebアプリケーションに簡単に統合できます。

CDNを使用したインストール

<script src="https://cdn.jsdelivr.net/npm/@konemono/nostr-web-components@latest/dist/nostr-web-components.iife.js"></script>

主要なコンポーネント

  • nostr-container: 子コンポーネントにリレー設定を提供
  • nostr-note: 個別のNostr投稿を表示
  • nostr-profile: ユーザープロフィールを表示
  • nostr-list: 複数の投稿をリスト表示
  • nostr-naddr: naddr形式のNostrエンコード識別子をもとにイベントリストを表示（例：kind 10003/30003のブックマークやリスト）。表示形式やソート順、ページネーションなどを属性で制御可能。
